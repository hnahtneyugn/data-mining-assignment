##  Assignment 1: Stock Data Analysis (HPG)

This notebook focuses on **exploratory data analysis (EDA)** and **data preprocessing** using real-world stock and financial data of **Hòa Phát Group (HPG)**.

###  Dataset Description

Two datasets were used:

* `Simplize_HPG_PriceHistory_20250315.xlsx`: Historical stock price data

  * Skipped 5 rows, read 1001 rows
* `Simplize_HPG_FinancialIndicator_20250315.xlsx`: Financial indicators of HPG

  * Skipped 6 rows, read 63 rows

###  Main Tasks

1. **Data Loading & Cleaning**

   * Remove null values and handle missing data
   * Convert date columns for proper time-series analysis
2. **Exploratory Data Analysis**

   * Visualize stock price trends over time using `matplotlib` and `seaborn`
   * Explore relationships between financial indicators
3. **Feature Engineering**

   * Compute derived metrics such as returns, moving averages, or growth rates
4. **Data Visualization**

   * Create trend plots, histograms, and correlation heatmaps
